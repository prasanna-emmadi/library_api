# library_api

Library rest api for books

## Getting started

### Development

- Start server and watch for changes
  - npm run watch

### Production

- Start the server
  - npm start

## Structure

- The project is structured as the following
  - routes
    - Has the routers
  - models
    - Has the db models
  - controllers
    - Has the api handler
  - middleware
    - Has the middleware like handling the passport jwt
